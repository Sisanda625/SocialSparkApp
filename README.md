The Social Spark application is a native Android app developed in Kotlin using Android Studio, 
designed to provide context aware social interaction suggestions. The app accepts user input representing times of day 
(e.g morning, afternoon, dinner, night) and uses a when statement to map these inputs to predefined social "sparks" actionable suggestions that
encourage users to reach out to friends, family, or colleagues. The user interface consists of an EditText field for input, a Button to generate
suggestions, a TextView to display results, and a Reset button for clearing the UI. Error handling is implemented using Toast messages that provide
friendly, constructive feedback for invalid inputs. The project is version-controlled with GitHub and includes a GitHub Actions workflow that
automatically builds the APK on every push to the main branch, ensuring continuous integration and build verification. This implementation demonstrates
proficiency in Android fundamentals, Kotlin syntax, UI design, and CI/CD practices.
